# Measuring Lived Racial Experience

R-based analysis pipeline for studying lived racial experience using the 2016 National Asian American Survey (NAAS).

## Project Structure

- `src/`: analysis notebooks (`.Rmd`)
- `functions/`: reusable modeling/plotting helpers
- `raw_data/`: source data files (`.dta`, not tracked)
- `processed_data/`: intermediate data products
- `outputs/`: figures/tables generated by notebooks
- `paper/`: manuscript (`paper.qmd`) and rendered files (`.pdf`, `.docx`)

## Analysis Pipeline

Run notebooks in order:

1. `src/01_data_cleaning.Rmd`
2. `src/02_imputation.Rmd`
3. `src/03_factor_analysis.Rmd`
4. `src/04_reg_analysis.Rmd`
5. `src/05_interaction_analysis.Rmd`
6. `src/06_table1_descriptive.Rmd` (optional descriptive table notebook)

## Current Modeling Notes

- Survey weight used: `pidadjweight` (carried as `weight`)
- Weighted and unweighted models are both estimated where relevant
- Construct scoring uses loading-based retained items from factor analysis
- Linked-fate results: race-interaction models + marginal effects
- Policy results: AAPI-only additive multivariable regressions comparing:
  - `discrimination`
  - `micro_aggression`
  - `democrat`
  - `republican`
